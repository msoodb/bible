    - [Fuzzing]
2. Go through the API documentation.
    - [Swagger]
    - [Stoplight]
    - [Readme]

3. Setup environments.
    - [BurpSuite]
        - scope
        - proxy
    - [Postman]
        - import api docs into postman
        - setup proxy to burpsuite
3. Determine the attack surface includes all of the inputs and outputs of the API. 
    - API calls
    - URL parameters
    - Headers
    - Cookies
    - Web responses
    - File uploads
    - API keys
